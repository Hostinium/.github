# Hostinium Public Presence

This repository manages the public-facing GitHub presence for Hostinium.

It contains the organization profile content, portfolio-safe product summaries, and supporting documentation used to present the company professionally without exposing private implementation details.

## What This Repository Includes

- the organization profile shown on the Hostinium GitHub page
- public portfolio briefs for private products
- positioning material for product-safe public presentation
- repository guidance for company-facing GitHub content

## Public Entry Points

- Organization page: [Hostinium on GitHub](https://github.com/Hostinium)
- Company website: [hostinium.com](https://hostinium.com)
- Benthium showcase: [Hostinium/Benthium-Showcase](https://github.com/Hostinium/Benthium-Showcase)

## Content Principles

- present products at the business and architecture-summary level
- protect implementation details, internal workflows, and delivery-specific know-how
- keep public messaging consistent with a long-term product company identity
- use GitHub as a trust surface, not as a leak surface

## Repository Layout

- profile/README.md: organization profile content rendered on the Hostinium page
- docs/PORTFOLIO.md: public portfolio map and visibility structure
- docs/products/: safe public product briefs for private offerings

## Intended Audience

- prospective clients and partners
- technical evaluators reviewing company credibility
- hiring, collaboration, and procurement stakeholders
- visitors who need a quick understanding of what Hostinium builds
